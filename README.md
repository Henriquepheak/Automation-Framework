# Automation Framework
An open-source Automation Framework.

## Installation
```bash
git clone https://github.com/mytechnotalent/af.git
```

## Setup Environment
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run
```bash
python af_cli.py
```
